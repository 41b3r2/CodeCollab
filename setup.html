Firebase Setup Guide

CodeCollab uses Firebase Realtime Database for real-time collaboration. Follow these steps to configure your free Firebase project.

1
Create a Firebase Project

Go to console.firebase.google.com and sign in with your Google account.

Click "Add project", give it a name (e.g., codecollab), optionally disable Google Analytics, and click "Create project".

2
Register a Web App

On your project dashboard, click the Web icon (</>) to add a web app to your Firebase project.

Give it a nickname (e.g., codecollab-web), skip Firebase Hosting, click "Register app".

You'll see your firebaseConfig object — copy those values into the form below.

3
Enable Realtime Database

In the left sidebar: Build → Realtime Database → Create database.

Choose a location closest to your users, then select "Start in test mode" (you can restrict rules later).

Recommended Database Rules (paste in Firebase console → Rules tab):
// Firebase Realtime Database Rules { "rules": { "rooms": { "$roomId": { // Anyone can read/write rooms (change for production) ".read": "true", ".write": "true", "chat": { ".indexOn": ["ts"] } } } } }
Test mode allows anyone to read/write. For production, add authentication and restrict rules.
